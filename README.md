# AstulJurondo Home

Cloudflare Pages-ready official worldview website prototype for AstulJurondo / 居容台.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: `main`

The first release is a static React/Vite site. Future dynamic content can be moved to Cloudflare D1 for structured records and R2 for gallery/media assets.
